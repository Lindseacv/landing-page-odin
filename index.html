Basic stitches to learn first

This waterfall uses backstitches and long-and-short stitches to create borders and to fill in large areas French knots accent the piece.
This spiderweb uses splitstitch with a single thread to achieve a delicate and continuous look.
This plant uses a woven stitch for the basket using multiple threads. The leaves are done in backstitches to outline with straight stitches to fill.
Once you've mastered different stitches, play with thread color and textures to create interesting pieces such as these fireflies!
